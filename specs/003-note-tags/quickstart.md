# Quickstart: Seamless note tags

1. Log in, open any note **Edit** page.
2. Confirm a **Tags** row appears between the title and the editor with chips for each `#tag` in the body.
3. Type `#test-tag` in the body — chip list updates shortly after typing stops.
4. Click **×** on a chip — the corresponding `#token` is removed from the body.
5. Use **Add tag…**, type `another`, press Enter — body gains `#another` (normalized) and the chip appears.
6. Save; open **Notes** list and filter by tag — behavior matches previous notes with tag filter.
