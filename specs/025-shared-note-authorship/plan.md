# Implementation Plan: Shared Note Authorship & Indicators

**Branch**: `025-shared-note-authorship` | **Date**: 2026-05-15 | **Spec**: [spec.md](spec.md)  
**Input**: Feature specification from `specs/025-shared-note-authorship/spec.md`

## Summary

Surface the git commit author on version history entries and the note reader page, and add outgoing/incoming share indicators to the notes lists. All author data is already stored in git commits (via the existing `CommitFileAs` mechanism); this feature wires that data into the owner-save flow and surfaces it in templates. Share indicators use a single aggregating query against the existing `note_shares` table — no DB schema changes.

### Step 6 — Add `ListShareCountsForOwner` model function

**File**: `backend/internal/models/shares.go`

```go
// ListShareCountsForOwner returns a map of note ID → active collaborator count
// for all non-archived notes owned by userID.
func ListShareCountsForOwner(db *DB, userID int64) (map[int64]int, error) {
    rows, err := db.Query(`
        SELECT ns.note_id, COUNT(*) AS collab_count
        FROM note_shares ns
        JOIN notes n ON n.id = ns.note_id
        WHERE n.user_id = ? AND n.archived = 0
        GROUP BY ns.note_id`,
        userID)
    if err != nil {
        return nil, err
    }
    defer rows.Close()
    result := make(map[int64]int)
    for rows.Next() {
        var noteID int64
        var count int
        if err := rows.Scan(&noteID, &count); err != nil {
            return nil, err
        }
        result[noteID] = count
    }
    return result, rows.Err()
}
```

### Step 9 — Add `SharedNoteHistoryGET` handler and route

**File**: `backend/internal/handlers/shares.go`

New handler:
```go
func (h *Handler) SharedNoteHistoryGET(w http.ResponseWriter, r *http.Request) {
    viewerID := userIDFromSession(r)
    ownerUsername := chi.URLParam(r, "username")
    slug := chi.URLParam(r, "slug")

    note, role, ownerUser, err := models.GetNoteForViewer(h.db, viewerID, ownerUsername, slug)
    if err != nil || note == nil || role == "" {
        http.NotFound(w, r)
        return
    }

    // Pagination
    page, perPage := parsePagination(r, 20)
    offset := (page - 1) * perPage

    relPath := fmt.Sprintf("%d/%s.md", ownerUser.ID, slug)
    versions, err := versioning.Log(h.notesDir, relPath, perPage+1, offset)
    if err != nil {
        http.Error(w, "history error", http.StatusInternalServerError)
        return
    }

    hasMore := len(versions) > perPage
    if hasMore {
        versions = versions[:perPage]
    }

    data := map[string]any{
        "Note":          note,
        "OwnerUsername": ownerUsername,
        "Versions":      versions,
        "Page":          page,
        "PrevPage":      page - 1,
        "NextPage":      page + 1,
        "HasMore":       hasMore,
        "PerPage":       perPage,
    }
    h.render(w, r, "shared/history.html", data)
}
```

Note: `models.GetNoteForViewer` may need to return the owner's User struct (or at least owner ID). Check current signature and adjust if needed. The owner's userID is already accessible via `note.UserID`.
